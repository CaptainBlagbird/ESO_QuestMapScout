Read me

The add-on automatically collects information about quests/maps for improving the add-on Quest Map.
After a while (e.g. after completing an alliance/dlc), please send me the data:

  1. Close the game
  2. Find the log file Documents\Elder Scrolls Online\<server>\SavedVariables\QuestMapScout.lua
    (<server> = live, liveeu or pts)
  3. Upload the file to my DropBox (no account required / email doesn't have to be valid)
  4. Delete the file from the SavedVariables folder

To see what data exactly the add-on collects, see the file example_log.lua where I added comments (behind the --) for each item.
